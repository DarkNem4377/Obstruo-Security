How it works

It answers the question before
the internet hears it.

Every website starts as a DNS question: "where is this site?" Obstruo answers that question on your own machine — before any connection exists.

01 · INTERCEPT

LISTEN

A Windows Service becomes your machine's DNS resolver. Every browser and every app asks it first — because Windows asks it.

02 · DECIDE

MATCH

Each query is checked against the blocklist stored on your disk. No third party is consulted. Allowed queries pass to the upstream DNS you already use.

03 · REFUSE

DENY

Blocked domains never resolve. The page never opens, the image never loads — the connection never exists.

Tamper resistance

Turning it off requires the key.
Nothing else works.

During setup you choose a PIN, a password, and get a one-time recovery code. Windows Hello works too, if your hardware supports it. From that point on, the credentials are the only door.

DISABLE

Pausing or disabling protection requires your credentials. Three wrong attempts triggers an escalating lockout that gets longer every time.

KILL

If the service stops — crash, kill, power loss — the filter fails closed. Nothing opens when the guard goes down. A watchdog restores normal DNS only through a verified recovery path.

BYPASS

Browsers that try to use their own encrypted DNS are stopped by firewall rules and enforced browser policies. The resolver on this machine stays the only route.

DATA

The database holding your blocklist and logs is encrypted with a key unique to this machine. Copying the file elsewhere yields nothing readable.

RECOVER

Lost your PIN and password? The one-time recovery code from setup is the only way back in. Lose that too, and the honest answer is: a clean uninstall — by design.

FAIL-CLOSED

Most filters fail open — crash them and everything unlocks. Obstruo is built the other way: if Obstruo stops, the filter does not.

Honest limits

What Obstruo does not do.

A security tool you can trust is one that states its edges plainly. Here are Obstruo's.

NO PAGE READING

Filtering is by domain, not page content. A site is either refused or resolved — Obstruo does not inspect what loads inside allowed sites.

NO SURVEILLANCE

No screenshots, no keylogging, no message reading. The only record kept is which domains were blocked, and when — encrypted, on this machine.

NO PHONES

Obstruo protects Windows machines. Phones, tablets, and consoles need their own controls.

NO MAGIC

An administrator with unlimited time and full system access can eventually defeat any local software. For children, give them a standard (non-admin) Windows account. For yourself, it is engineered friction — and friction works.

Questions

Details for the reader
who reads details.

Does it work in Chrome, Firefox, Edge — any browser?

Yes. Obstruo doesn't live inside a browser — it answers DNS for the whole machine. Every browser and every app that resolves a domain goes through it, including private/incognito windows.

What about DNS-over-HTTPS bypass?

Browsers that try to use their own encrypted DNS are blocked by firewall rules against known DoH endpoints, and browser policies for Chrome, Edge, and Firefox disable secure-DNS overrides. Queries stay on the local resolver.

Does it phone home?

No. There is no account, no cloud service, no telemetry endpoint. Filtering, logging, and credential checks all happen on your machine.

Can it be uninstalled?

Yes — with your credentials. The uninstaller requires the PIN or password set during installation, removes everything cleanly, and restores your original DNS settings.

What if I lose my PIN, password, and recovery code?

Then there is no back door — deliberately. A filter with a support hotline that unlocks it isn't tamper-resistant. The setup wizard shows the recovery code exactly once; store it somewhere safe.

What data do you collect?

None. Block events are logged to an encrypted database on your own disk so you can review them, with a retention window you control. We never see them. We couldn't if we wanted to.

Does it slow down my internet?

No meaningful difference. DNS lookups happen in memory on your own machine — allowed queries pass straight to the upstream resolver you already use.

What does it cost?

Nothing — Obstruo is free during early access. No license key, no trial timer, no locked "premium" tier. Right now the goal is users and feedback, not revenue.

Why not just edit the hosts file?

A hosts file has no lock on the door: anyone with admin rights can undo it in Notepad in ten seconds, it fails open, nothing maintains the list, and nothing stops encrypted-DNS bypass. Obstruo adds the parts that matter: credentials, a fail-closed watchdog, DoH blocking, a curated list, and encrypted logs.

Does it filter the whole machine or just one user account?

The whole machine. DNS is answered system-wide, so every Windows account, browser, and app on the device is covered by the same rules.

Will my child see it running?

Obstruo is deliberately quiet: no branded block pages, no popups — a blocked site simply never loads. The dashboard opens only with your credentials.

Windows Home or Pro?

Both. The browser policies Obstruo enforces are applied through the registry, which works on Home as well as Pro — no Group Policy editor needed.
